Source Code

Source code is available on the master branch, or by rooting around in the subdirectories of these pages. Most of it is written in Julia, but there is some stuff, especially early on, that was implemented in MATLAB. I highly recommend using Julia over MATLAB, Python, or C++. See below for links that I found helpful:
